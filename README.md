# pinachigong
## 실제 운영 되고있는 피자나라치킨공주 데이터를 통해서 유의미한 인사이트 및 도움이 되고자 분석</br>
성과: 데이터 기반 휴일 정하기, 배달경로 추이 확인 , 인수전후 매출 비교 확인 , 배달경로 150일간의 변화 예측모델 구현 및 시각화
